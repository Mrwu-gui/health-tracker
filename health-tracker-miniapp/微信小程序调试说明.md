# 微信小程序调试说明

## 报错：未找到 pages/login/index.wxml

**原因**：在微信开发者工具里导入的是「项目根目录」，而根目录只有 `.vue` 源码，没有编译后的 `.wxml` 文件。

**正确做法**：在微信开发者工具中导入的目录必须是 **编译输出目录**：

```
项目根目录/unpackage/dist/dev/mp-weixin
```

## 步骤

1. **编译小程序**（二选一）  
   - 在项目根目录执行：`npm run dev:mp-weixin`（监听改动自动编译）  
   - 或执行：`npm run build:mp-weixin`（单次编译）  
   - 或使用 HBuilderX：运行 → 运行到小程序模拟器 → 微信开发者工具  

2. **在微信开发者工具中**  
   - 导入项目 / 打开项目  
   - 目录选择：**`unpackage/dist/dev/mp-weixin`**（不要选项目根目录）

3. 之后修改 `pages/**/*.vue` 后，保存即可自动编译，开发者工具会自动刷新。
